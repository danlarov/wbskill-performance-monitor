# 🚀 WBSkill Performance Monitor

[![WordPress](https://img.shields.io/badge/WordPress-6.9+-blue?logo=wordpress)](https://wordpress.org)
[![PHP](https://img.shields.io/badge/PHP-7.4+-777bb4?logo=php)](https://php.net)

**WBSkill Performance Monitor** is a powerful, all-in-one WordPress plugin that helps you identify slow plugins, optimize your database, monitor hooks and cron jobs, and get actionable insights to speed up your website.

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 📊 **Plugin Performance Analysis** | Identify exactly which plugins are slowing down your site with precise timing measurements |
| 🗄️ **Database Optimization** | Clean up revisions, transients, auto-drafts, trash, and orphaned meta data with one click |
| 🎨 **Asset Analysis** | Scan and analyze CSS, JS files loaded on your site |
| 🔌 **Hook & Filter Monitor** | Find excessive hooks that impact performance |
| ⏰ **Cron Job Inspector** | See all scheduled tasks and their impact on performance |
| 📈 **Performance Scoring** | Get a clear 0-100% performance score for your entire site |
| 🤖 **Automated Scans** | Schedule regular performance scans (daily, weekly, monthly) |
| 📄 **Exportable Reports** | Generate detailed reports in JSON, CSV, or HTML format |
| 🌍 **Multilingual** | Full support for English and Russian languages |
| 🆓 **100% Free** | No premium upsells, no hidden costs |

---

## 📋 Why Choose This Plugin?

Most performance plugins only show you basic metrics. **WBSkill Performance Monitor** goes deeper:

1. **Pinpoints exact plugins** causing slowdowns — not just general suggestions
2. **Provides actionable recommendations** tailored to your specific setup
3. **Safe one-click optimizations** with confirmation dialogs
4. **Privacy-focused** — no data sent to external servers
5. **Beautiful dashboard** with modern cards and real-time statistics

## 📥 Installation

### Method 1: Upload via WordPress Admin (Recommended for beginners)

1. Download the latest ZIP archive from [Releases](https://github.com/danlarov/wp-performance-monitor/releases)
2. Log in to your WordPress admin panel
3. Go to **Plugins → Add New**
4. Click the **Upload Plugin** button at the top of the page
5. Click **Choose File**, select the downloaded ZIP archive
6. Click **Install Now**
7. After installation, click **Activate Plugin**

### Method 2: Manual Installation via FTP (For advanced users)

1. Download the latest ZIP archive from [Releases](https://github.com/danlarov/wp-performance-monitor/releases)
2. Extract the ZIP file to your computer
3. Connect to your hosting server using an FTP client (FileZilla, Cyberduck, etc.)
4. Navigate to the WordPress plugins directory: `/wp-content/plugins/`
5. Upload the extracted folder `wbskill-performance-monitor` to this directory
6. Log in to your WordPress admin panel
7. Go to **Plugins → Installed Plugins**
8. Find **WBSkill Performance Monitor** and click **Activate**

## 🚀 Quick Start Guide

1. After activation, go to **Performance Monitor** in your WordPress admin menu
2. Click **Start Scan** to analyze your site
3. Review the results:
   - **Dashboard** – See your overall performance score
   - **Scanner** – Analyze plugins, database, assets, hooks, and cron jobs
   - **Database** – Clean up revisions, drafts, trash, and transients
   - **Reports** – Export performance reports
   - **Tools** – Test performance, view system info, clear caches
   - **Settings** – Configure auto-scans, email reports, and more

## 📊 What You Can Clean

| Item | Description | Safe to Delete? |
|------|-------------|-----------------|
| Post Revisions | Every change saved to your posts | ✅ Yes |
| Auto Drafts | Automatically saved drafts | ✅ Yes |
| Trashed Items | Posts and comments in trash | ✅ Yes |
| Expired Transients | Temporary expired data | ✅ Yes |
| Orphaned Meta Data | Meta data belonging to deleted content | ✅ Yes |

## ❓ Frequently Asked Questions

### Is this plugin really free?

**Yes!** WBSkill Performance Monitor is completely free and always will be. If you find it useful, consider donating to support development.

### Will scanning slow down my site?

No. Scans run efficiently and only when manually triggered or on a schedule. They won't impact your visitors' experience.

### Is it safe to optimize the database?

**Yes.** Our optimization tools only remove safe-to-delete data like revisions, trashed items, and expired transients.

### Does it work with caching plugins?

**Absolutely!** WBSkill Performance Monitor works alongside any caching plugin (WP Rocket, W3 Total Cache, LiteSpeed Cache, etc.).

### Does it work with WooCommerce?

**Yes.** The plugin is fully compatible with WooCommerce and analyzes all WooCommerce plugins and extensions.

== Description ==

Comprehensive performance monitoring tool that identifies slow plugins and optimizes database.

= Key Features =

* **Plugin Performance Analysis** - Identify exactly which plugins are slowing down your site with precise timing measurements
* **Database Optimization** - Clean up revisions, transients, and orphaned data with one click
* **Asset Analysis** - Scan and analyze CSS, JS, and image files for optimization opportunities
* **Hook & Filter Monitor** - Find excessive hooks that impact performance
* **Cron Job Inspector** - See all scheduled tasks and their impact
* **Performance Scoring** - Get a clear 0-100% performance score for your entire site
* **Automated Scans** - Schedule regular performance scans
* **Exportable Reports** - Generate detailed JSON reports for developers
* **Multilingual** - Full support for English and Russian languages

= Why Performance Monitor? =

Most performance plugins only show you basic metrics. Performance Monitor goes deeper:

1. **Pinpoints exact plugins** causing slowdowns, not just general suggestions
2. **Provides actionable recommendations** tailored to your specific setup
3. **Safe one-click optimizations** with rollback capability
4. **Completely free** with no premium upsells
5. **Privacy-focused** - no data sent to external servers

= Use Cases =

* **Site Owners** - Find out why your site is slow without hiring a developer
* **Agencies** - Monitor client sites and provide optimization reports
* **Developers** - Debug performance issues during development

== Installation ==

1. Upload the `wp-performance-monitor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Performance Monitor' in your admin menu
4. Click 'Start Scan' to analyze your site
5. Review recommendations and apply optimizations

== Frequently Asked Questions ==

= Is this plugin really free? =

Yes! Performance Monitor is completely free and always will be. If you find it useful, consider donating to support development.

= Will scanning slow down my site? =

Scans run efficiently and only when manually triggered or on a schedule. They won't impact your visitors' experience.

= Is it safe to optimize the database? =

Yes, our optimization tools only remove safe-to-delete data like revisions, trashed items, and expired transients.

= Does it work with caching plugins? =

Absolutely! Performance Monitor works alongside any caching plugin.
